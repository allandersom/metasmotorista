# Features

Camada reservada para funcionalidades completas do sistema.

Modulos planejados:

- `lancamentos.js`: formulario, historico e exclusao de lancamentos
- `rankings.js`: rankings diarios/mensais e exportacao
- `dom-feriados.js`: painel e relatorio de domingos/feriados
- `projecao.js`: graficos e metricas de projecao
- `rotas.js`: rota do dia e planilha de rotas
- `cadastro-motoristas.js`: cadastro, edicao e status de motoristas
- `faltas-atestados.js`: relatorios de faltas e atestados
